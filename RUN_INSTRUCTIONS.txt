================================================================================
                              TROUBLESHOOTING
================================================================================

Q: "Connection failed. Is the broker running?"
A: Start the broker FIRST in Terminal 1, then start consumer/producer

Q: Broker stops immediately
A: This is now FIXED! The broker runs continuously.

Q: Can't bind to port 9092
A: Another process might be using port 9092. Check with:
   lsof -i :9092
   Kill that process or change the port in the code.

Q: Nothing happens when I send messages
A: Make sure all 3 components are running:
   1. Broker (Terminal 1)
   2. Consumer (Terminal 2)  
   3. Producer (Terminal 3)
